# "MacGyver in the maze" (E. MINCARELLI - 07/11/2020)

# Quel est le but de ce jeu ?
Ce jeu a pour but d'aider MacGyver à sortir d'un labyrinthe dans lequel il est enfermé.
Pour cela, il faudra guider le personnage de MacGyver dans ce dédale, avec les touches de direction, afin de récupérer 3 objets disséminés dans le labyrinthe et qui seront nécessaires pour endormir le gardien qui en interdit la sortie.

# Comment installer le jeu ?
Pour installer le jeu, il vous suffit de télécharger l'ensemble des fichiers et dossiers contenu dans le dépôt Github suivant : https://github.com/MINCARELLI13/Projet-MacGyver.git

# Comment lancer le jeu ?
Pour commencer à jouer, il n'y a plus qu'à lancer le fichier "main.py".

# Quels sont les ressources nécessaires au fonctionnement du jeu ?
Ce jeu nécessite l'utilisation de l'application Python.exe ainsi que des librairies et modules suivants : Pygame, os et random.
